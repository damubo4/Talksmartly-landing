Preguntas frecuentes

¿Necesito tener conocimientos previos para inscribirme?

@if (isTooltipQ1) {

No necesitas tener ningún conocimiento previo en el idioma, ya que ofrecemos cursos desde el nivel introductory, los cuales están enfocados en personas que aún no cuentan con ningún acercamiento del idioma.

}

¿Debo comprar libros para los cursos?

@if (isTooltipQ2) {

No, ya que nosotros diseñamos un material exclusivo de acuerdo a tus objetivos y necesidades; de este modo el docente te guiará en tu proceso de aprendizaje con contextos de acuerdo a tus intereses laborales o personales.

}

¿Puedo crear y escoger el número de clases que desee?

@if (isTooltipQ3) {

¡Claro que sí! Si ningún plan se acomoda a lo que deseas, puedes contactarnos para crear juntos uno para ti.

}

¿Los cursos están disponibles para todas las edades?

@if (isTooltipQ4) {

Nuestros cursos conversacionales están diseñados para trabajar con personas desde los 17 años, pero nuestros cursos personalizados son para todas las edades.

}

¿Qué diferencia hay entre los cursos grupales e individuales?

@if (isTooltipQ5) {

Nuestros cursos grupales están enfocados a la práctica conversacional, con clases semi-personalizadas de hasta máximo 4 estudiantes, en donde se discuten temas de actualidad y de interés. Por otro lado, las clases personalizadas son sesiones uno a uno con el docente. En este curso creamos material exclusivo para ti donde se enfoca en tus metas y en tus necesidades para que alcances resultados de manera más eficiente.

}

¿Puedo obtener un certificado al finalizar el curso?

@if (isTooltipQ6) {

Te otorgamos certificados de participación y cumplimiento de nuestros cursos; sin embargo, este certificado no tendrá validez académica, pero si deseas puedes formarte con nosotros en la preparación para exámenes internacionales como IELTS, TOEFL, MIT, Linguaskill, entre otros. Nosotros te asesoramos en todo tu proceso de certificación.

}

¿Cómo puedo saber qué nivel de idioma tengo?

@if (isTooltipQ7) {

En el momento que adquieres cualquiera de nuestros cursos o planes, tendrás una entrevista totalmente gratuita donde un profesor experto evaluará el nivel en el que te encuentras.

}

¿Cómo puedo pasar de un nivel a otro?

@if (isTooltipQ8) {

Para avanzar de un nivel a otro necesitas haber finalizado el curso del nivel en el que te encuentres, o si consideras haber alcanzado el nivel antes de culminarlo, puedes, en cualquiera de los dos casos, debes solicitar una entrevista de reclasificación.

}
